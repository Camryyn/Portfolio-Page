Projects

eMotive

eMotive is a AI sentimental analysis based website coded primarily in Python. To achieve accurate and usable results the LLM was trained using a database of movie reviews to help the AI differentiate positive and negative word associations. This lead to a product that takes user inputted sentences and outputs the corresponding emotional values scaled by percent of core emotions. This site also features a SQL database to save inputs and continue training model for future accuracy.

Skylines

Skylines is a API based website coded in Javascript and utilizes NoSQL to connect to MongoDb. The purpose for this website is to be able to serve a better way of giving recommendations of flights that may be affordable and have concise interface for filtering out the best flight for the user. This site features a star review system, a dynamic and easily readable display for output plane tickets and a user friendly filter system for a more user-specific plane ride.
