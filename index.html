Skylines

Skylines is a API based website coded in Javascript and utilizes NoSQL to connect to MongoDb. The purpose for this website is to be able to serve a better way of giving recommendations of flights that may be affordable and have concise interface for filtering out the best flight for the user. This site features a star review system, a dynamic and easily readable display for output plane tickets and a user friendly filter system for a more user-specific plane ride.
